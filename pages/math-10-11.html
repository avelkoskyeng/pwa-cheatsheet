Функции

Функция — зависимость переменной y от переменной x, при которой одному значению х соответствует только одно значение y.

Пример зависимости
Автомобиль движется со скоростью x км/ч и проезжает в день y км. Если увеличить скорость x, расстояние y тоже увеличится. х — независимая переменная функции, аргумент y — зависимая переменная функции.

Области функций

Все допустимые значения x — область определения функции.
Все допустимые значения y — область значений функции.

Так как функция обозначает зависимость, её можно отобразить на графике, где x — значение по оси абсцисс, y — значение по оси ординат.

Свойства функций

Область определения — D(x) Все значения, которые принимает независимая переменная — аргумент х.
Область значения — E(y) Все допустимые значения, которые принимает зависимая переменная — функция у.
Промежутки возрастания и убывания f(x) возрастает, если большему значению аргумента x соответствует большее значение функции f(x). f(x) убывает, если большему значению аргумента x соответствует меньшее значение функции f(x).
Промежутки знакопостоянства Все значения аргумента х, при которых функция принимает положительные (y > 0) или отрицательные (y < 0) значения.
Нули функции Значения аргумента х, при которых значение функции равно нулю (у = 0).
Чётность функции f(x) — чётная, если f(–x) = f(x), график чётной функции симметричен относительно оси Oy. f(x) — нечётная, если f(–x) = –f(x), график нечётной функции симметричен относительно начала координат.
Наибольшее и наименьше езначение функции Наибольшее значение функции — это такое число М = f(x0), что для любого x верно: f(x) ≤ f(x0). Наименьшее значение функции — это такое число m = f(x0), что для любого x верно: f(x) ≥ f(x0).

Виды функций и их графики

Линейная функция: y = kx + b

График — прямая. Если k > 0, функция возрастает; если k < 0 — убывает, b — точка пересечения с осью ординат.
Функция возрастает. k > 0
Функция убывает. k < 0
Пересечение с осью Oy выше начала координат. b > 0
Пересечение с осью Oy ниже начала координат. b < 0
